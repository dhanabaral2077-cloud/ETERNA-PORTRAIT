# **App Name**: Pet Masterpiece

## Core Features:

- Portfolio Gallery: Display a high-quality, zoomable image gallery of custom pet portraits, filterable by pet type.
- Commission Process: Explain the pet portrait commission process, including steps for providing pet photos and choosing portrait styles.
- Pricing Information: Present clear and comprehensive pricing for different portrait sizes, styles, and add-ons.
- Client Testimonials: Showcase positive feedback and reviews from satisfied clients.
- Contact/Order Form: An integrated contact/order form that enables the user to send clear instructions for the pet's photo and the preferred drawing style.
- AI Style Suggestion: Based on a customer's description of what kind of artwork is desired, use a tool to suggest relevant pieces of the artist's existing portfolio that are aligned with the customer's taste. 

## Style Guidelines:

- Primary color: Terracotta (#E2725B), an earthy tone reflecting the warmth and artistry of pet portraits.
- Background color: Soft Beige (#F5F5DC), a light and neutral background for a warm and inviting atmosphere.
- Accent color: Deep Forest Green (#228B22) to highlight key elements.
- Font: 'Alegreya', a serif with an elegant, intellectual, contemporary feel.
- A one-page layout that leverages a grid system.
- Subtle transitions and hover effects for a smooth and engaging user experience.
